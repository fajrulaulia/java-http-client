## Library
- http://hc.apache.org/downloads.cgi

## Before Start
make sure you system have installed JDK / OpenJDK

## how to run 
- check `run.sh`
- run
  ``` bash
   sudo chmod +x run.sh
   ./run.sh
  ```
- make sure internet connect to world
